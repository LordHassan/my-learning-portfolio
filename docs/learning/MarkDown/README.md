<h1  style="text-align:center; text-decoration: underline; ">MarkDown</h1>

#### **介紹**

Markdown 是一種「易讀易寫」的輕量級標記語言，它使用簡單的純文字格式來編寫文件，並能輕易地將其轉換成其他格式，如 HTML、PDF 等。它專注於內容本身，語法簡潔，使用符號（如 # 表示標題，\* 表示列表）來標記格式，非常適合撰寫網路內容、筆記和文件。

---

#### **主要特色**

- 簡潔易讀： 語法簡單，即使是純文字也能看出文件結構，易於閱讀和編寫。

- 專注內容： 讓使用者專注於寫作內容，而非排版細節，效率更高。

- 可移植性高： 檔案是純文字格式 (.md)，可在任何文字編輯器中編輯，跨平台使用。

- 格式轉換： 可輕鬆轉換為多種格式，如 HTML、PDF、Word 文檔等，滿足不同發布需求。

---

#### **常見應用**

- 網路寫作： 許多網站和平台（如 GitHub、Reddit、Stack Exchange 等）廣泛使用 Markdown 來撰寫說明文件、論壇貼文等。

- 程式碼文件： 許多程式碼儲存庫（如 GitHub）使用 Markdown 編寫 README 文件。

- 筆記與電子書： 可用於個人筆記和撰寫電子書。

- 文件與報告： 許多工具（如 Google 文件）已支援或提供 Markdown 啟用選項。

- AI 輸出： 大語言模型輸出的文件也常用 Markdown 格式，以支援簡單的排版需求。
