> ## 常用功能快捷鍵

<dl>
    <dt>移動</dt>
    <dd>移動段落：游標在段落上 <code>alt + uparrow / downarrow </code></dd>
    <dd>複製並移動段落： 游標在段落上 <code>shift + alt + uparrow / downarrow </code></dd>
    <dt>插入</dt>
    <dd>插入空行至下方： 游標在段落上 <code>ctrl + enter </code></dd>
    <dd>插入空行至上方： 游標在段落上 <code>ctrl + shift + enter</code> </dd>

    <dd>複製並移動段落： 游標在段落上 shift + alt + uparrow / downarrow </dd>
    <dd>複製並移動段落： 游標在段落上 shift + alt + uparrow / downarrow </dd>

</dl>
